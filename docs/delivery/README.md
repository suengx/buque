# 一期验收推进

## 交付物

- **验收工作簿**：[`补雀_一期交付验收确认表.xlsx`](./补雀_一期交付验收确认表.xlsx)（可发给计划/运营填写）
- **重新生成**：`cd backend && uv run python scripts/generate_delivery_workbook.py`
- **生产部署**：[`deploy/README.md`](../deploy/README.md)

## 建议节奏（3 步）

### 第 1 步：演示会（约 30 分钟）

按工作簿 `00_使用说明` 中的议程，在生产或预发 URL 上演示，现场填写 **`03_功能走查`**。

### 第 2 步：业务试用（3–5 个工作日）

- 计划部日常使用日报、预警、监控助手
- 每周填写 **`05_试运行记录`**
- 并行催收 **`excel_baseline.xlsx`**（见 `backend/fixtures/README.md`）

### 第 3 步：评审签字

- 有 Excel 基准后，IT 跑 `compare_excel.py`，结果填入 **`02_验收指标`**、**`04_Excel一致率`**
- 确认 **`01_一期范围确认`**
- **`06_交付签字`** 由计划负责人组织
